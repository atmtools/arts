ARTS
====

**pyarts** is the Python interface for the
`Atmospheric Radiative Transfer Simulator <https://www.radiativetransfer.org>`_
(ARTS). It provides an interactive interface to the ARTS engine for running radiative transfer simulations.
This is the documentation for ARTS version |release|. Other versions, scientific publications, and related tools
are available on the `ARTS website <https://www.radiativetransfer.org>`_.
